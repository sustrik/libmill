LIBMILL
=======

Libmill is a library that introduces Go-style concurrency to C.

Libmill works with x86-64, gcc and Linux/OSX.

Building and installing from source
-----------------------------------

```
$ ./autogen.sh
$ ./configure
$ make
$ make check
$ sudo make install
```
