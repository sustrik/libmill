Prerequisites

  • Install Ruby
  • Install libuv from here: https://github.com/joyent/libuv

The former is needed at compilation time, the latter at both compilation time and runtime.

Getting the code

For now, download the source code from GitHub:

https://github.com/sustrik/mill

Installation

    $ ./autogen.sh
    $ ./configure
    $ make
    $ sudo make install
